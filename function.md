# **Функциональные требования**
## ***Клиент поставщик (Программный агент):***
+ Формирование отчётов:
 + Преобразование таблиц из 1С:Предприятие в pdf файлы
 + Присвоение уникального ID для поставки клиенту получателю
 + Отправка отчётов на сайт
+ Разграничение пользователей по правам доступа:
 + Клиент-получатель
 + Пользователи разных компаний
 + Клиент-поставщик
+ Обработка запросов на оказание услуги:
 + Приём запроса в неформальном виде
 + Приём официального запроса
## ***Администратор:***
+ Обработка списка клиентов-пользователей:
 + Создание
 + Удаление
 + Изменение
## ***Клиент-получатель:***
+ Авторизация:
 + Сверка логин\пароль
 + Двухфакторная авторизация
+ Отчёты:
 + Сортировка:
  + По указанному периоду
  + По специализации:
   + Отчёты по реализации
   + Отчёты по платежам
   + Отчёты выполненных работ
 + Сохранение отчёта в файл pdf
 + Печать отчёта
 + Вывод отчёта на экран
+ Формирование заказа акта сверки из 1С
+ Формирование запроса на выполнение работ
+ Хранение проведённых актов сверки:
 + Сортировка по указанному периоду
 + Сохранение акта сверки в файл pdf
 + Печать
 + Вывод на экран
+ Автоматический выход при бездействии
